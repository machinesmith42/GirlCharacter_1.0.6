Version: 1.0.6
Date: 16. 06. 2020
  Bugfix:
    - Fixed that corpse of default character would be used instead of the Gear Girl corpse.
  Changes:
    - Won't overwrite the default character anymore if a known character-selector mod is active.
    - "minime >= 0.0.14", "RitnCharacter > 0.0.3" and "CharSelect >= 0.18.1" will be automatically registered as known character-selector mods. The current version of "RitnCharacter" (0.0.3) is not yet recognized, so unless another of the above mods unlocks the new behavior, "GearGirlCharacter" should work with "RitnCharacter" as before!
  Locale:
    - Localized name/description of character and corpse. That's needed for minime, which allows to select a character by name.
  Credits:
    - Thanks to Pi-C on the Factorio mods website for coding this compability update!
